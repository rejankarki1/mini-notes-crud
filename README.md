# Mini Notes CRUD App

A simple CRUD notes application built using Express.js and EJS.

## Features
- Create notes
- Read all notes
- Update notes
- Delete notes

## Technologies Used
- Node.js
- Express.js
- EJS
- Method Override
- UUID

## What I Learned
- RESTful routing
- CRUD operations
- req.body and req.params
- Express middleware
- Dynamic EJS rendering
- Form handling
- Method override for PATCH and DELETE

## Routes Used

GET /notes  
POST /notes  
PATCH /notes/:id  
DELETE /notes/:id  

## Future Improvements
- Connect MongoDB
- Add authentication
